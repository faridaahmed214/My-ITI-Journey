# Maximum (Ternary)

وظيفة الفولدر:

- تقارن بين رقمين وتعرض أكبر واحد باستخدام تعبير ثلاثي (ternary).

الملفات:

- `maximumWzTernery.html`
- `maximumWzTernery.js` — يطلب رقمين عبر `prompt` ويعرض `alert` بالرقم الأكبر.

إزاي تشغل:

- افتح `maximumWzTernery/maximumWzTernery.html` في المتصفح واتبع الـ prompts.

ملاحظات:

- الكود الحالي يستخدم ternary داخل `if` بشكل مباشر — شغّال لكن ممكن تبسيطه أو تتأكد من تحويل المدخلات إلى أعداد (`parseInt`).

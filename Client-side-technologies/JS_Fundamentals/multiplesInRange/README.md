# Multiples In Range

الغرض:

- يعرض الأرقام في رينج اللي هي مضاعفات 3 ومضاعفات 5، وبيرجع مجموع الأرقام اللي هي مضاعفات 3 أو 5.

الملفات:

- `multiplesInRange.html`
- `multiplesInRange.js` — يطلب `start` و`end`، يجمع ويطبع النتائج في `console`.

كيفية الاستخدام:

- افتح `multiplesInRange/multiplesInRange.html` ثم ادخل قيم البداية والنهاية عبر prompts.
- افتح DevTools → Console لرؤية القوائم والمجموع.

ملاحظات:

- القوائم مخزنة كسلاسل نصية مفصولة بمسافات.
- يمكن تعديل الكود لاستخدام مصفوفات أو DOM لعرض النتائج بشكل أجمل.
